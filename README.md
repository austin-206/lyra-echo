This repository contains the MkDocs documentation, including architecture, governance, integration notes, and configuration.

Documentation site:
https://austin-206.github.io/lyra-echo/

## Local Development

Activate virtual environment:
source .venv/bin/activate

Serve docs locally:
mkdocs serve

Deploy to GitHub Pages:
mkdocs gh-deploy
