# Change Log

## Purpose
Track and summarize all system, documentation, and configuration changes across Lyra Echo releases. Use this file for formal versioned updates; append new entries chronologically.

---

## Template for New Entry
```
## [vX.Y.Z] — YYYY‑MM‑DD
### Added
- New features or hardware components introduced.

### Changed
- Updates to configurations, docs, or behaviors.

### Fixed
- Bug fixes, latency optimizations, and corrections.

### Deprecated
- Features or endpoints scheduled for removal.

### Security
- Patches or mitigations related to security controls.
```
